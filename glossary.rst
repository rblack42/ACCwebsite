Glossary
########

..  glossary::

    Continuous Integration
        A system where project code is automatically checked out from a
        :term:`source code control system` installed and tested each time
        changes are integrated into the code repository.

    Source Code Control System
        A tool used to manage the history of project files.

    Open-Source
        Software projects released under a license that requires including
        source code so developers can extend the code.

    Test-Driven Development
        A software development where tests for new features are written before
        the feature is implemented, then the code is written to pass the test.
